Guide

Najpierw należy sprawdzić wersję Angular'a przez polecenie ng v i zainstalować kompatybilną wersję primeNG poleceniem npm install primeng.

Następnie należy przejść do pliku angular.json i umieścić w: "styles":[ "node_modules/primeng/resources/themes/lara-light-blue/theme.css", "node_modules/primeng/resources/primeng.min.css", ] oraz przejść do pliku style.css lub style.scss i umieścić w nim: import "primeng/resources/themes/lara-light-blue/theme.css"; import "primeng/resources/primeng.css";

Następnie w pliku TypeScript komponentu w którym chcemy umieścić Accordion należy dodać zaimportować primeng/accordion a później w imports dodać AccordionModule
